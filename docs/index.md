# SleepLab Documentation

Welcome to the SleepLab documentation site. SleepLab is an open-source health data platform designed with local-first, privacy-focused principles.

---

## 1. Key Features

- **Automated Data Ingestion:** Advanced pipelines for importing ResMed CPAP `DATALOG` directories and binary EDF files.
- **Wearable telemetry overlays:** Correlate therapy parameters with sleep stages, heart rate, and oximetry data.
- **Interactive Dashboards:** Rich, beautiful visualizations of long-term health trends and detailed daily sleep metrics.
- **AI-Powered Insights:** Automated clinical summaries of sessions and therapy trends powered by cloud or local LLMs.
