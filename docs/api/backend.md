# SleepLab 2.0 import diagnostics

Authenticated 2.0 diagnostics endpoints:

- `GET /imports/runs?limit=25`: recent durable imports with machine identity,
  adapter, status, capabilities, warnings/errors, counts, and source-file
  count.
- `GET /imports/runs/{run_id}`: one run with its complete source manifest and
  file dispositions.
- `GET /imports/machines`: durable CPAP machine identities, validation/support
  state, session count, and import count.
- `GET /imports/machines/{machine_id}/settings`: versioned settings snapshots.

`POST /upload/source/{upload_id}/finish` now returns `import_run_id` when an
approved ResMed plan starts. Detection-only sources remain blocked and do not
create a run.
